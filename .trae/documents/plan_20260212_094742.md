# Re-implement UI based on Game Rules

I will refactor the UI to strictly follow the "Flat Design" and "Purple Theme" specifications defined in `游戏规则.md`, removing the glassmorphism effects added in the previous iteration.

## 1. Visual Style Overhaul (Flat Design)
-   **Background**: Change from gradient/glass to solid **Light Purple/Gray** (`#F5F3FF` / `bg-violet-50`).
-   **Cards**: Switch from translucent/blurred to **Solid White** (`#FFFFFF` / `bg-white`) with soft shadows (`shadow-xl`).
-   **Primary Color**: Ensure consistent use of **Purple** (`#8B5CF6` / `bg-violet-500`) for primary actions and highlights.
-   **Typography**: Use **Dark Slate** (`#1E293B` / `text-slate-800`) for main text to ensure high contrast.
