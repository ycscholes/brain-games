# 奇趣图形记忆

## 训练目标

通过 N-Back 图形回忆训练短时记忆、持续更新和限时判断。

## 入口与状态

- 路由：`/pages/memory-challenge/index`
- `gameId`：`memory-challenge`
- 状态：开始、初始记忆、答题、结算

## 完整玩法

开始时依次展示 N 个初始图形，每个展示 1.5 秒。随后每轮展示一个新图形，玩家需要从四个选项中选出 N 轮之前出现的目标图形。答对后历史序列继续推进；答错或倒计时结束时本局结束。

## 模式与难度

答题时间可选 8、6、4、2 秒，对应简单至专家。记忆跨度可选 1-Back 至 4-Back。任一维度达到第 3 档或第 4 档时，本局按困难积分计算，其余按普通积分计算。

## 游戏内计分

每次答对根据时间与记忆跨度组合获得 2 至 4 分。页面内分数最多累计到 40 分；答错和超时不再继续。

## 积分管线

基础转换率为 1.0。结算通过 `getAwardedPoints("memory-challenge", score, difficulty)` 计算积分，并由 `addPointsToPet()` 写入宠物余额。

## 训练记录

完成后记录 `mode: "T{时间档}M{记忆档}"`、积分难度和最终分数。各难度组合分别保存最高分，旧 `memory` 训练 ID 在汇总时与标准 ID 合并。

## 实现与测试

页面逻辑位于 `index.tsx`，图形素材位于 `src/assets/shapes/`。当前没有独立游戏逻辑测试；积分换算和旧 ID 兼容由 `tests/unit/trainingStorage.test.ts` 覆盖。
