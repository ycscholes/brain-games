# 追踪任务

## 训练目标

训练动态场景中的多目标视觉追踪和持续注意。

## 入口与状态

- 路由：`/pages/multiple-object-tracking/index`
- `gameId`：`multiple-object-tracking`
- 状态：开始、目标预览、移动追踪、选择、结算

## 完整玩法

每轮先高亮目标圆圈，随后所有圆圈恢复相同外观并在场地内移动、碰撞和反弹。移动结束后，玩家必须选出全部目标。全部正确才进入下一轮，任一选择错误则本局结束。

## 模式与难度

普通模式从 2 个目标开始，预览 1.2 秒、追踪 5 秒。困难模式从 3 个目标开始，预览 0.9 秒、追踪 5.5 秒且初始速度更高。每轮成功后目标数最多增加到 4 个，移动速度持续提升。

## 游戏内计分

每完整通过一轮获得 1 分。部分选对不计分。

## 积分管线

基础转换率为 3.0。主动返回开始页时会以 `interrupted` 状态记录当前成绩并结算积分；选择错误时以 `completed` 状态结算。

## 训练记录

记录分数、积分、难度和完成状态。普通与困难分别保存最高分，旧 `mot` ID 在汇总时与标准 ID 合并。

## 实现与测试

场地、运动、碰撞、计时和选择逻辑集中在 `index.tsx`。当前没有独立追踪逻辑测试；积分与旧 ID 兼容由 `tests/unit/trainingStorage.test.ts` 覆盖。
