# 词语拼盘

## 训练目标

训练汉字重组、语义匹配和限时语言加工。

## 入口与状态

- 路由：`/pages/word-scramble/index`
- `gameId`：`word-scramble`
- 状态：开始、8 题答题、反馈、结算
- 首页状态：当前隐藏，但路由与记录能力保留

## 完整玩法

每局固定 8 题。页面给出词义提示和打乱的汉字，同时混入无关汉字；玩家根据提示重组目标词，并从四个词语选项中确认答案。每题有独立时限，难度和题序越高，展示与作答时间越短。

## 模式与难度

普通题库以两个汉字的常用词为主，干扰字较少。困难题库包含三至五个汉字的词组，干扰字更多、提示延迟更长、时限更短。

## 游戏内计分

- 答对基础分：4
- 2.2 秒内答对：额外 1 分
- 当前连击至少为 2：额外 1 分
- 答错或超时：0 分

## 积分管线

基础转换率为 1.0。结算调用共享积分与宠物余额函数。

## 训练记录

完成 8 题后记录分数、积分、难度和总耗时。普通与困难分别保存最高分。

## 实现与测试

`gameLogic.ts` 负责题库、打乱、干扰字、时限和计分；`index.tsx` 负责交互与存储。`tests/unit/wordScrambleGameLogic.test.ts` 覆盖题目生成、选项和计分。
