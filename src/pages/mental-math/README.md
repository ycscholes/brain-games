# 速算挑战

## 训练目标

训练基础计算速度、准确性和连续作答稳定性。

## 入口与状态

- 路由：`/pages/mental-math/index`
- `gameId`：`mental-math`
- 状态：开始、答题、结算

## 完整玩法

每题显示一道整数算术题和多个答案选项。限时模式在固定时间内尽量多答对；闯关模式连续作答，答错立即结束。限时模式答错后仍会进入下一题。

## 模式与难度

固定训练内容包括 10 以内加减、20 以内进退位、百以内加减、乘法口诀、万以内加减和 100 以内四则混合。前四项按普通积分计算，后两项按困难积分计算。

自定义训练允许组合加、减、乘、除和数字范围。有效分数系数由范围与最难运算共同决定，系数达到 1.5 时按困难积分计算。

## 游戏内计分

基础成绩为答对题数。固定训练直接使用答对题数；自定义训练会先乘以配置系数，再将有效分数交给公共积分系统。

## 积分管线

基础转换率为 1.0。积分难度由所选训练内容自动决定，最终仍通过 `getAwardedPoints()` 和 `addPointsToPet()` 处理。

## 训练记录

固定内容的模式格式为 `{timed|death}:{阶段 ID}`。自定义内容额外记录运算组合、范围和系数。最高分按游戏模式和训练阶段分别保存。

## 实现与测试

`mathStages.ts` 负责阶段配置、自定义系数和题目生成，`index.tsx` 负责游戏流程。`tests/unit/mentalMathStages.test.ts` 覆盖阶段、自定义配置和题目约束。
