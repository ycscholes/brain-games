# 逆向猜拳

## 训练目标

通过“赢、平、输”指令切换训练反应速度、规则保持和抑制控制。

## 入口与状态

- 路由：`/pages/rock-paper-scissors/index`
- `gameId`：`rock-paper-scissors`
- 状态：开始、连续答题、结算

## 完整玩法

每题显示电脑手势和目标结果。玩家需要选择能让自己“赢”“平”或“输”的手势，而不是固定追求胜利。答对后立即进入下一题；答错或单题倒计时结束时本局结束。

## 模式与难度

四档难度的单题时限分别为 5、4、3、2 秒。第 1、2 档按普通积分计算，第 3、4 档按困难积分计算。

## 游戏内计分

四档难度每题分别获得 2、3、3、4 分，页面内分数最多为 40。答错会清空连击并结束本局，连击用于结算展示，不额外加分。

## 积分管线

基础转换率为 1.0。结算统一调用 `getAwardedPoints()` 和 `addPointsToPet()`。

## 训练记录

训练记录使用 `mode: "D{难度档}"`。每档难度分别保存最高分，旧 `rps` ID 在训练汇总中与标准 ID 合并。

## 实现与测试

题目生成、答案判断、计时和持久化集中在 `index.tsx`。当前没有独立游戏逻辑测试；公共积分行为由 `tests/unit/trainingStorage.test.ts` 覆盖。
