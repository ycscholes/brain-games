# 24 点

## 训练目标

训练四则运算组合、心算搜索和表达式规划。

## 入口与状态

- 路由：`/pages/twenty-four/index`
- `gameId`：`twenty-four`
- 状态：开始、限时答题、结算

## 完整玩法

每轮给出四张数字牌。玩家必须各使用一次，通过加、减、乘、除和括号组成结果为 24 的表达式。题目生成时会验证存在解法。玩家可以换题或查看参考解法；使用提示的当前题不计分。

## 模式与难度

普通模式 90 秒，困难模式 60 秒。两个模式的牌面数字都只能为 0 至 10。

## 游戏内计分

每独立解出一题获得 2 分。错误表达式、未完整使用四张牌或提示题不计分。结算同时展示解题数和游戏得分，最高纪录与训练记录使用游戏得分。

## 积分管线

基础转换率为 2.0。解出一题获得 2 游戏分，并换算为 4 宠物积分；困难模式继续应用共享 1.5 倍倍率和 60 分封顶。

## 训练记录

结算记录 `mode: "90s"` 或 `mode: "60s"`、难度、完整时长、分数和积分。普通与困难分别保存最高分。

## 实现与测试

表达式求解、生成和求值位于 `gameLogic.ts`。`tests/unit/twentyFourGameLogic.test.ts` 覆盖数字边界、题目可解性、单题分值和表达式求值；积分行为由 `tests/unit/trainingStorage.test.ts` 覆盖。
