# 数字广度记忆

## 训练目标

训练短时记忆容量、顺序保持和准确复述。

## 入口与状态

- 路由：`/pages/digit-span/index`
- `gameId`：`digit-span`
- 状态：开始、逐位展示、输入、结算

## 完整玩法

系统逐位展示随机数字串，展示结束后玩家用数字键盘输入完整序列。回答正确后长度增加 1；回答错误时本局结束。

## 模式与难度

普通模式从 3 位开始，每位展示 1 秒。困难模式从 4 位开始，每位展示 0.8 秒。

## 游戏内计分

最终成绩等于成功记住的最大数字长度。若起始轮即失败，成绩为 0。

## 积分管线

基础转换率为 3.0。普通和困难分别使用共享倍率与积分上限。

## 训练记录

完成后记录最终长度、积分和难度。普通与困难分别保存最高分。

## 实现与测试

序列生成、展示计时和输入流程集中在 `index.tsx`。当前没有独立游戏逻辑测试；积分行为由 `tests/unit/trainingStorage.test.ts` 覆盖。
