# 星链回响

## 训练目标

训练视觉路径工作记忆、顺序保持和空间定位。

## 入口与状态

- 路由：`/pages/number-order/index`
- `gameId`：`number-order`
- 状态：开始、准备、闪现、作答、回放、结算

## 完整玩法

每局固定 8 张星图。系统在多个星点中依次闪现一条星链，玩家记住出现顺序，并在闪现结束后按原顺序点亮对应星点。答错后立即停止本题，并回放正确路径和本题得分。

当前实现只有“按闪现顺序复现”的星链回响规则，不使用历史设计中的数字升降序、颜色排序或亮度排序。

## 模式与难度

普通模式每图 4 至 6 个星点，需要记忆 3 至 5 个步骤，闪现间隔由 920 毫秒逐步降至 730 毫秒。困难模式每图 5 至 8 个星点，需要记忆 4 至 7 个步骤，间隔由 720 毫秒降至 520 毫秒。

## 游戏内计分

只有完整复现路径才计分。长度 3、4-5、6 以上的路径基础分分别为 3、4、5 分。每连续完整通过 3 题增加 1 点组合奖励，单题组合奖励最多 2 分。错误路径为 0 分。

## 积分管线

基础转换率为 1.0。普通和困难分别使用共享倍率与积分上限。

## 训练记录

结算记录分数、积分、难度和总耗时。普通与困难分别保存最高分。

## 实现与测试

`gameLogic.ts` 负责星点布局、闪现序列、路径验证、回放文本和计分；`index.tsx` 负责阶段和渲染。`tests/unit/numberOrderGameLogic.test.ts` 覆盖生成范围、路径判断和计分。
